Proof

Since the set \(D\) of discontinuity of \(f\) is negligible, for any \(\epsilon > 0\) there exists an open set \(G\) in \([a, b]\) such that it covers \(D\) and \(|G| < \epsilon\text{.}\) For every point \(x\) in the closed set \([a, b]\setminus G\text{,}\) \(\osc(f)(x)=0\text{,}\) so \(f\) is a continuous function on this closed set. It can be extended to a continuous function \(c\) on \([a, b]\) such that \(\max_{[a, b]}|c| \le \max_{[a,b]}|f|\text{.}\) Then this \(c\) satisfies our requirements.
