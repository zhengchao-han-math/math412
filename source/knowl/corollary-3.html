Corollary 3.6. Approximation Property of Functions in \(\cR[a, b]\).

Any Riemann integrable function \(f\in \cR[a, b]\) lies in \(L^{p}[a, b]\) for any \(1\le p < \infty\text{,}\) and for any \(\epsilon > 0\) there exists a continuous function \(c\in C[a, b]\) and an open set \(G\) in \([a, b]\) such that \(|G| < \epsilon\text{,}\) \(f=c\) on \(G^{c}\text{,}\) and \(\max_{[a, b]}|c| \le \max_{[a,b]}|f|\text{.}\)
