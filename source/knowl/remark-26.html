Remark 3.2.3.

Condition (3.2.1) implies that \(g(t)\) is continuous at \(x\text{.}\) Note that if \(g'(x)\) exists, then (3.2.1) is satisfied at \(x\text{.}\) In fact, (3.2.1) is satisfied at \(x\) if both the left derivative \(g'(x-)\) and the right derivative \(g'(x+)\) exist.
If \(g(t)\) has both a right limit \(g(x+)\) and a left limit \(g(x-)\) at \(x\) and \(g(x+)\ne g(x-)\text{,}\) we can rewrite \(S_N[g]( x)\) as
\begin{equation*} S_N[g]( x)= \frac{1}{2\pi} \int_{0}^{\pi}\left[ g(x-t)+g(x+t)\right] D_{N}(t)\, dt, \end{equation*}
so
\begin{align*} \amp S_N[g]( x)-\frac{g(x+)+g(x-)}{2}\\ = \amp \frac{1}{2\pi} \int_{0}^{\pi}\left[ g(x-t)-g(x-)+g(x+t)-g(x+)\right] D_{N}(t)\, dt. \end{align*}
If \(g(t)\) satisfies
\begin{equation*} \vert g(x-t)-g(x-) \vert, \vert g(x+t)-g(x+)\vert \le M t\text{ for $t>0$ near $0$}, \end{equation*}
we can make the same argument to show that \(S_N[g]( x)-\frac{g(x+)+g(x-)}{2}\to 0\) as \(N\to\infty\text{.}\)
Note that whether or not \(h(x; t)\) is integrable so as to apply Lemma 3.2.1 depends only on the the local behavior of \(g(t)\) near \(x\text{,}\) so whether or not \(S_N[g]( x)\) converges depends only on the local behavior of \(g(t)\) near \(x\text{.}\) This is known as the Riemann's localization theorem.
